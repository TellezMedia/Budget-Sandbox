Your money, your data

Sign in with Google. Your budget lives in your own Google Sheet, only you can see it.

First-time setup required. Before signing in, you need a Google OAuth Client ID.

1. Go to console.cloud.google.com and create a project.
2. Enable Google Sheets API and Google Drive API.
3. Under Credentials, create an OAuth 2.0 Client ID (Web application type).
4. Add your GitHub Pages URL to Authorized JavaScript origins.
5. Copy the Client ID and paste it into CLIENT_ID at the top of this file.

Hosting: Push this file to a GitHub repo, enable GitHub Pages under Settings, and your app is live for free.

Welcome to Budget

Your personal finance tracker. Data lives in your own Google Sheet. Nobody else can see it.

1
Go to the Accounts tab
Open your banking app, grab your current balance for each account, and enter it here. This is your starting point.
2
Add your recurring bills
Go to the Bills tab and enter anything you pay on a regular schedule. Rent, subscriptions, utilities, phone bill.
3
Start logging expenses
Every time you spend money, log it in the Expenses tab. The app deducts it from your balance automatically.
Pro tip: Run the Balance Transactions check on the Accounts tab weekly to make sure the math is staying clean.
